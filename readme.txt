This plugin automatically adds HTML anchor tags to plain text links and email addresses embedded in the content of posts and pages.

== Description ==

This plugin automatically adds HTML anchor tags to plain text links and email addresses embedded in the content of posts and pages (WordPress already does this by default for comments).

In the plugin configuration page, you can choose to open the links in a new window or in the current window.

== Installation ==

1. Upload `autolink.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit the Autolink options page to manage the plugin options
